READ ME
---
This program simplifies the process of setting a timer to shutdown/restart/logoff your computer. This is a PC version. 

---

The pack containing 1 program:
Shutdown Timer

After you run the program, there will always be a new text and a new batch file as side product of the operation. It is ok to delete them

Should you require to create an icon on desktop, make a shortcut, or copy the program to the desire location.

---
HOW TO USE

First, select how long you want to set to shut down the computer with the number of hours, minutes and seonds.
Then, choose the option (Shutdown, Restart, or Log Off). A timer and a loading bar will appear.

If you wish to cancel and perform another operation, simply close the program and open it again.
