READ ME
---

The pack containing 1 program:
Shutdown Timer

After you run the program, there will always be a new text and a new batch file as side product of the operation.

Should you require to create an icon on desktop, make a shortcut. 

---
HOW TO USE

First, select how long you want to set to shut down the computer with the number of hours, minutes and seonds.
Then, choose the option (Shutdown, Restart, or Log Off). A timer and a loading bar will appear.

If you wish to cancel and perform another operation, simply close the program.
